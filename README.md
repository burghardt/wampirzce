Wampirzce
---------

Symulacja Wampirzcow-Rozpustnikow

How to build it?
----------------

To compile it you will need:
 * SDL
 * freeglut
 * automake

Install `SDL-devel`, `freeglut-devel` and `automake` (Fedora):

    # sudo dnf install SDL-devel freeglut-devel automake

Compile `wampirzce` with:

    # sh autogen.sh
    # sh configure
    # make

Enjoy!
